## About Me

안녕하세요. 웹, 모바일, AI를 폭넓게 경험하며 **문제를 구조적으로 이해하고 구현으로 연결하는 소프트웨어 개발자 이민우**입니다.

현재는 **Vision SW 연구원**으로 커리어를 시작하며, 이미지 기반 인식과 자동화 시스템 구현에 관심을 넓혀가고 있습니다.  
저는 특정 기술 하나에 머무르기보다, 서비스의 데이터 흐름과 사용자 경험, 그리고 실제 동작하는 시스템 구조를 함께 고민하는 개발을 지향합니다.

주요 관심 분야는 다음과 같습니다.

- **Web Frontend**: React, TypeScript 기반의 웹 서비스 개발
- **Cross-platform App**: Flutter, React Native 기반 모바일 앱 개발
- **LLM & AI**: LLM 파인튜닝, LoRA, DPO 기반 정렬 연구
- **Vision AI**: 이미지 처리 및 비전 기반 자동 인식 시스템
- **Data-driven Service**: API 연동, 상태 관리, 사용자 흐름 설계

## Experience & Interests

### Frontend Engineering
React와 TypeScript를 기반으로 웹 서비스를 개발하며, API 연동, 상태 관리, 반응형 UI, 사용자 흐름 최적화에 관심을 가지고 있습니다.  
TanStack Query와 Zustand를 활용해 서버 상태와 클라이언트 상태를 분리하고, 유지보수 가능한 프론트엔드 구조를 설계하는 경험을 쌓았습니다.

### Cross-platform App Development
Flutter와 React Native를 활용해 모바일 앱을 개발했습니다.  
OAuth, FCM, 네이티브 API 연동, 앱스토어 배포 등 실제 서비스 운영에 필요한 기능 구현과 배포 과정을 경험했습니다.

### LLM Research & Fine-tuning
Gemma 기반 LLM 파인튜닝 프로젝트를 수행하며, 비정형 데이터 수집·정제, Q&A 데이터셋 구축, LoRA 기반 경량 파인튜닝, DPO 기반 선호도 정렬 연구를 경험했습니다.

### Vision AI
마이크로피펫 분주량 자동 인식 프로젝트를 통해 이미지 기반 측정 및 자동 인식 문제에 관심을 가지게 되었습니다.  
현재는 Vision SW 연구원으로서 비전 기반 소프트웨어와 자동화 시스템 구현 역량을 확장하고 있습니다.
